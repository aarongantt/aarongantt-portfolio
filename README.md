# Aaron Gantt - Steadicam & Camera Operator

Professional portfolio website for Aaron Gantt, SOC member and Steadicam operator.

## Deployment

This site is deployed on Vercel and accessible at [aarongantt.com](https://aarongantt.com)

## Local Development

Simply open any `.html` file in a browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000

## Structure

- `index.html` - Home page
- `reel.html` - Demo reel
- `works.html` - Portfolio work
- `clients.html` - Client logos
- `equipment.html` - Equipment list
- `bio.html` - Biography
- `booking.html` - Contact/booking
- `images/` - All site images
- `fonts/` - Custom fonts
